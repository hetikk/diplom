Слёрм выпустил продвинутый практический курс по Docker
Блог компании Southbridge,
Учебный процесс в IT,
Карьера в IT-индустрии



Команда учебного центра Слёрм представила новый курс по Docker для тех, кто сталкивается с Docker в работе: системных администраторов (в т.ч. начинающих девопсов), разработчиков (особенно пишущих микросервисы), тестировщиков, QA-инженеров. На курсе речь пойдет о повседневных практических задачах: подробности.


Слёрм — компания, где преподают исключительно практикующие специалисты. Курс по Docker готовили инженеры Southbridge: Марсель Ибраев, Елизавета Михно, Александр Швалов. Добрый десяток DevOps-инженеров из других компаний протестировал курс. По их фидбеку мы внесли и до сих пор вносим улучшения. Любой фидбек приветствуется и идет в работу.


Обучение разделено на две части: базовую и продвинутую. Базовая — что из себя представляет Docker, какие есть основные команды и абстракции, как работать с Docker-compose. Уроки продвинутого уровня объяснят, как работает Docker изнутри и какие технологии использует, как устроена и работает сеть Docker. Вы познакомитесь с best practice и приемами работы с Docker для разных языков программирования.


Курс свежий, записан в июле 2020 года. Когда выйдет новая версия Docker или изменятся best practice, мы обновим версию курса для всех владельцев.


В курс входят видеозаписи лекций, Git-репозиторий с материалами и стенд для практики от партнёра Selectel.


По базовым знаниям есть требования:
Обучение платное. Регистрация на курс по Docker здесь.
